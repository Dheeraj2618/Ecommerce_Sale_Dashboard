# 🛒 Ecommerce Sales Dashboard

## 📌 Project Overview

The **Ecommerce Sales Dashboard** is an interactive data visualization project built using **Microsoft Power BI**. The dashboard provides valuable insights into ecommerce sales performance by analyzing revenue, product categories, payment methods, countries, and sales trends.

The dataset was cleaned and prepared using **Microsoft Excel** before being imported into Power BI for visualization.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor revenue and units sold.
- Identify top-performing products and categories.
- Compare sales across different countries.
- Analyze payment methods.
- Generate business insights through interactive dashboards.

---

## 🛠️ Tools Used

- **Microsoft Excel** – Data Cleaning & Preparation
- **Microsoft Power BI** – Dashboard Design & Data Visualization

---

## 🧹 Data Cleaning (Excel)

The following data cleaning steps were performed:

- Removed duplicate records
- Checked for missing values
- Corrected inconsistent data
- Standardized column names
- Formatted data for analysis

## 📈 Key Insights

- Total Revenue: **44M**
- Total Units Sold: **7.38M**
- Dhaka is the top-performing division.
- Card is the most preferred payment method.
- Beverage products generate the highest revenue.
- Bangladesh contributes the highest revenue among all countries.
